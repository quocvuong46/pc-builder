CÔNG NGHỆ SỬ DỤNG
========================================

- Language: Java 11
- GUI: Java Swing
- Database: SQLite
- Build Tool: Maven
- Design Patterns: Singleton, Decorator

========================================
TÍNH NĂNG CHÍNH
========================================

1. Chọn linh kiện máy tính (CPU, RAM, VGA, etc.)
2. Tính toán tổng giá cấu hình
3. Thêm dịch vụ bổ sung (Decorator Pattern):
   - Bảo hành mở rộng
   - Dịch vụ lắp ráp
   - Cài đặt phần mềm
   - Giảm giá
4. Quản lý database với Singleton Pattern
5. Hiển thị chi tiết cấu hình
